# polygon.nft\_contracts

All Polygon erc721 contracts & subset of erc1155 contracts that contain NFTs.

| Column Name  | Data Type         |
| ------------ | ----------------- |
| `address`    | CHARACTER VARYING |
| `is_erc721`  | BOOLEAN           |
| `is_erc1155` | BOOLEAN           |
