# prices.btc.coinbase

| `timestamp` | TIMESTAMP |
| `usd_high`  | DECIMAL   |
| `usd_low`   | DECIMAL   |
| `usd_open`  | DECIMAL   |
| `usd_close` | DECIMAL   |
