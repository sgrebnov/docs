# prices.btc.gemini

HLOC values for BTC at 1 minute granularity from Gemini.

| Column Name | Data Type |
| ----------- | --------- |
| `timestamp` | TIMESTAMP |
| `usd_high`  | DECIMAL   |
| `usd_low`   | DECIMAL   |
| `usd_open`  | DECIMAL   |
| `usd_close` | DECIMAL   |
